Requirements
------------

Kdyby/Money requires PHP 5.3.2 or higher.

- [Nette Framework](https://github.com/nette/nette)


Installation
------------

The best way to install Kdyby/Money is using  [Composer](http://getcomposer.org/):

```sh
$ composer require kdyby/money:@dev
```
